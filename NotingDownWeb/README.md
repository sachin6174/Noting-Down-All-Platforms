# NotingDownWeb

A simple web client for the Noting Down application. This is a vanilla HTML/CSS/JavaScript application that connects to the ANotingDownServer API.

## Features

- User authentication (login/signup)
- Create, edit, delete, and view notes
- Responsive design
- Session persistence
- Real-time note management

## Setup

1. Install dependencies:
```bash
npm install
```

2. Make sure the ANotingDownServer is running on http://localhost:3001

3. Start the web client:
```bash
npm start
```

The client will be available at http://localhost:3000

## Usage

1. Open your browser and go to http://localhost:3000
2. Sign up for a new account or login with existing credentials
3. Create, edit, and manage your notes
4. Notes are automatically saved to your account

## Requirements

- Node.js (for the http-server)
- ANotingDownServer running on port 3001
- Modern web browser with JavaScript enabled
